# 🤖 AI Sentiment Analyzer

## 📌 About This Project
An AI powered sentiment analyzer that detects
positive, negative and neutral emotions from
any text input using Python and TextBlob!

## 🎯 What This Project Does
- ✅ Detects Positive sentiment 😊
- ✅ Detects Negative sentiment 😞
- ✅ Detects Neutral sentiment 😐
- ✅ Gives exact polarity score
- ✅ Interactive mode — analyze your own text!

## 🛠️ Tools & Technologies Used
- Python 🐍
- TextBlob Library
- Natural Language Processing (NLP)

## 💡 How It Works
TextBlob analyzes the text and gives a
polarity score between -1 and +1
- Score > 0 = Positive 😊
- Score < 0 = Negative 😞
- Score = 0 = Neutral 😐

## 🗂️ Files in This Project
- sentiment_analyzer.py — Main Python code

## 🚀 How to Run
pip install textblob
python sentiment_analyzer.py
